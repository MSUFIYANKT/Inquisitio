### Database Tables

    -'users' ( User details)
    -'levels' (All the level details)
    -'teamlevels' (Level of team)

#### Users Table :

    #### coloumns
            - id
            - useremail
            - password
            - flag (Keep default 1)

#### levels table :

    #### coloumns
            - id
            - level (Level Number)
            - img (Question image name with extension, Eg : 1.jpg)
            - answer (Answer for the question)

#### teamlevels table :

    #### coloums
            - id
            - username
            - level
            - ans1
